yyg-cli 用于快速生成基于 vite + vue3 全家桶的脚手架项目。默认集成如下框架：


- vite 2
- vue 3
- typescript
- tsx
- vue router
- pinia
- element plus

## USAGE 用法

### Global Installation 全局安装

```shell
yarn global add yyg-cli
或
npm install yyg-cli -g
```

### Create Project 创建项目

```shell
yyg create <项目名>
```
如创建名为 `demo` 的项目，可使用命令：
```shell
yyg create demo
```
